# Worktable Playbook

## Qué puede hacer Project Brain ahora mismo

- resumir estado del proyecto
- recordar prioridades
- recomendar agente adecuado
- advertir riesgos recurrentes
- recordar cómo cerrar una tarea bien

## Qué preguntarle

- “¿Qué sigue ahora?”
- “¿Qué bloquea el release?”
- “¿Qué agente conviene para esta tarea?”
- “¿Qué non-negotiables debo revisar?”
- “¿Qué anti-patrones ya vimos en este repo?”
