# Truth And Signal Discipline

## Objetivo

Proteger el cerebro local de información falsa, inflada o poco reutilizable.

## Regla

Solo entra conocimiento si cumple al menos una de estas condiciones:

- evita un error repetido
- acelera una tarea frecuente
- mejora una decisión importante
- aclara una arquitectura
- mejora validación o calidad

## No entra al cerebro

- opiniones sueltas
- entusiasmo sin sistema
- frases motivacionales
- teoría sin efecto operativo
- “tips” sin contexto

## Protocolo de curación

1. detectar señal
2. verificar utilidad
3. clasificar como heurística, anti-patrón, checklist, prompt o template
4. guardar en el lugar correcto
5. referenciarlo desde la academia o el flujo
