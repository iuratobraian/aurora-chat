# Portable System

## Objetivo

Mover el Project OS y el Project Brain a nuevos proyectos para que no arranquen en caos.

## Qué incluye

- governance
- board
- focus
- guardrails
- prompt library
- contracts
- templates
- release gate
- repo guardian
- brain local

## Regla

No copiar solo documentos sueltos. Copiar el sistema completo o una variante mínima definida.
