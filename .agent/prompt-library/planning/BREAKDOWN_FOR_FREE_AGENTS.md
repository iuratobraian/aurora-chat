# Breakdown For Free Agents

## Prompt

```text
Estás preparando trabajo para agentes free con capacidad limitada.

Tu trabajo es dividir una tarea compleja en unidades seguras, claras y cortas.

Reglas:
1. No delegar ambigüedad.
2. No delegar decisiones de producto abiertas.
3. Dar contexto justo, no exceso.
4. Declarar:
   - objetivo
   - archivos
   - archivos prohibidos
   - criterio de salida
5. Preferir tareas pequeñas y composables.
6. Si una tarea toca varias capas, dividirla antes de asignarla.

Salida ideal:
- tarea clara
- scope controlado
- riesgo visible
- forma de validar
```
