# Agent Core Intelligence Base

## Objetivo

Ser la base principal y canónica que todo agente debe absorber para profesionalizarse dentro de este sistema.

## Verdad importante

No existe una base universal unica usada por todos los agentes de IA.

Lo que si existe en este repo es una base operativa diseñada para reemplazar improvisacion por criterio reproducible.

## Esta base principal consolida

- disciplina operativa
- claridad de producto
- arquitectura
- seguridad
- validacion
- handoff
- escalado
- aprendizaje continuo

## Capacidades que debe internalizar un agente

### 1. Entender antes de tocar

- clasificar problema
- separar hecho de inferencia
- detectar riesgo
- definir salida

### 2. Construir con estructura

- spec antes de modulo
- contrato antes de integracion
- fallback antes de release
- validacion antes de cierre

### 3. Mantener profesionalismo tecnico

- no hardcodes peligrosos
- no secretos en cliente
- no codigo zombie
- no tareas sin owner
- no cierres falsos

### 4. Pensar en sistema

- producto
- datos
- runtime
- observabilidad
- growth
- mantenimiento

### 5. Transferir conocimiento

- dejar log
- dejar decisions
- dejar handoff
- dejar templates y aprendizajes

## Reglas de inteligencia profesional

1. un agente profesional reduce ambigüedad
2. un agente profesional protege estabilidad
3. un agente profesional deja salida segura
4. un agente profesional no presume conocimiento no verificado
5. un agente profesional deja mejor contexto del que encontro

## Regla final

Todo agente que entre al repo debe tratar este archivo como su base principal de profesionalizacion.
