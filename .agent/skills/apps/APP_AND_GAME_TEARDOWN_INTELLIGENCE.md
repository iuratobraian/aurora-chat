# App And Game Teardown Intelligence

## Objetivo

Extraer conocimiento util de apps y juegos grandes para que el equipo pueda construir productos nativos con criterio incrustado.

## Que mirar en una app grande

### Producto

- promesa principal
- tiempo a primer valor
- arquitectura de superficies
- secuencia de descubrimiento
- retention hooks
- monetizacion

### UX

- onboarding
- empty states
- loading states
- error recovery
- accesibilidad
- mobile ergonomics

### Sistema

- identidad
- feed o home logic
- perfiles
- notificaciones
- search
- settings
- admin o internal tooling inferido

### Growth

- loops de sharing
- reactivation
- creators
- referrals
- pricing story

## Que mirar en un juego mobile

### Core

- 10-second loop
- 5-minute goal
- day-1 return
- dificultad
- feedback del input

### Meta

- progresion
- economia
- eventos
- live ops
- personalizacion
- social features

### Negocio

- monetizacion
- segmentacion
- tutorializacion
- reengagement

## Patrones transferibles

- shell muy claro
- data contracts fuertes
- experimentacion controlada
- observabilidad
- feature flags
- content operations
- tooling interno

## Antipatrones a evitar

- copiar UI sin entender el sistema
- copiar loops toxicos
- copiar complejidad sin necesidad
- basar decisiones en una sola fuente
