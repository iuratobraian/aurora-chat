# App Creation Levels

## Objetivo

Dar distintos niveles de creacion de apps para elegir el nivel correcto de ambicion y complejidad.

## Nivel 1: MVP util

- una promesa
- un flujo principal
- una metrica clave
- arquitectura minima

Ideal para:

- validar idea
- crear base
- llegar rapido a usuarios

## Nivel 2: Producto estable

- auth
- datos solidos
- observabilidad
- diseño consistente
- monetizacion inicial

Ideal para:

- primeras ventas
- primer equipo
- primeras integraciones

## Nivel 3: Plataforma escalable

- multiples modulos
- APIs
- dashboards
- automatizaciones
- roles y permisos
- capas de monetizacion

Ideal para:

- negocio en crecimiento
- varios tipos de usuario
- partners

## Nivel 4: Ecosistema

- marketplace
- creators
- integraciones externas
- intelligence layer
- apps satelite
- admin avanzado

Ideal para:

- startup grande
- producto compuesto
- expansion por verticales

## Nivel 5: Category leader

- platform thinking
- internal tooling
- experimentation
- growth engine
- data products
- ecosystem network effects

Ideal para:

- competir globalmente
- operar multiples lineas de ingreso

## Regla

No saltar de nivel sin cerrar el anterior.
