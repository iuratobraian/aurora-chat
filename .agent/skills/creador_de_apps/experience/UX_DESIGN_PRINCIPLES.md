# UX & Design Principles: Obsidian Ether

Nuestra identidad visual se basa en profundidad, claridad y proactividad.

## Principios de Diseño
1.  **Aero-Glass Depth**: Uso de desenfoque de fondo (backdrop-blur) y bordes semitransparentes para crear capas de información.
2.  **Asimetría Intencional**: Romper la rejilla tradicional para guiar el ojo hacia acciones críticas.
3.  **Micro-Animaciones**: Transiciones de 300ms-500ms para cada interacción de usuario.
4.  **Dark Mode First**: Diseñado originalmente para ambientes oscuros con acentos de color vibrantes (Primary: #2563eb).

## Guía de Componentes
- **Cards**: Bordes redondeados de 1.5rem (24px), glassmorphic effects.
- **Buttons**: Gradientes sutiles, sombras dinámicas al hover.
- **Typography**: Inter / Roboto, uso de pesos fuertes (font-black) para encabezados cortos.
