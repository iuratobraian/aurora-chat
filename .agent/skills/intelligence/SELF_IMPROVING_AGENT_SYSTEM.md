# Self Improving Agent System

## Objetivo

Hacer que los agentes del repo aprendan mejor, trabajen con más comodidad y acumulen inteligencia útil día a día.

## Verdad importante

La mejora no ocurre por magia ni auto-modificación ciega.

La mejora real ocurre cuando el sistema:

- registra trabajo
- destila aprendizaje
- convierte errores en reglas
- convierte éxitos en playbooks
- obliga a reutilizar lo aprendido

## Qué significa ser más inteligente en este repo

- entender más rápido
- decidir con menos ambigüedad
- cometer menos errores repetidos
- elegir mejor el siguiente paso
- dejar mejores handoffs
- usar mejor el conocimiento acumulado

## Pilares del sistema

### 1. Captura

Todo trabajo importante debe dejar:

- qué se intentó
- qué funcionó
- qué falló
- por qué falló
- qué conviene repetir

### 2. Curación

No todo hallazgo merece entrar al cerebro.
Solo entra lo que:

- se repite
- evita errores
- acelera trabajo
- mejora calidad

### 3. Destilación

Cada aprendizaje debe convertirse en una de estas formas:

- heurística
- anti-patrón
- checklist
- prompt
- template
- playbook

### 4. Reutilización

Si el conocimiento no vuelve al flujo, no sirve.

Debe reaparecer en:

- README
- academy
- templates
- prompts
- runbooks
- brain generated files

### 5. Revisión

Ningún agente debe “enseñarse” cambios críticos sin revisión humana o sin pasar por el Project OS.

## Loop de crecimiento diario

1. un agente ejecuta
2. registra aprendizaje
3. sincroniza el cerebro local
4. el conocimiento se clasifica
5. futuras tareas arrancan desde una base mejor

## Qué debe aprender cada día

- patrones de debugging
- riesgos de integración
- mejores secuencias de trabajo
- mejoras de producto
- respuestas al usuario más alineadas
- errores costosos a evitar

## Qué no debe aprender

- atajos inseguros
- suposiciones sin evidencia
- workarounds temporales como si fueran arquitectura
- copy-paste de soluciones sin contexto

## Regla final

Un agente se vuelve más fuerte cuando el sistema hace persistente su aprendizaje y obliga a reutilizarlo.
