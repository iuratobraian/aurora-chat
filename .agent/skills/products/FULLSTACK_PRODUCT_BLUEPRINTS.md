# Fullstack Product Blueprints

## Objetivo

Guiar construccion profesional de apps web y moviles desde cero.

## Capas

### Producto

- propuesta de valor
- usuario principal
- journeys
- monetizacion

### Frontend

- shell
- rutas o navegacion
- estados
- design system
- accesibilidad

### Backend

- servicios
- auth
- permisos
- jobs
- webhooks

### Datos

- entidades
- relaciones
- indices
- eventos
- auditoria

### Operacion

- logs
- alertas
- CI
- release

## Regla de construccion

Un modulo nuevo debe nacer con:

- contrato
- owner
- fallback
- validacion

## Prioridades

1. estabilidad
2. claridad
3. seguridad
4. velocidad
5. sofisticacion
