# Website Creator App Platform

## Objetivo

Definir una futura app capaz de crear sitios web completos, funcionales y profesionales a partir de ideas simples.

## Lo que debe hacer

- convertir una idea corta en spec web
- clasificar tipo de sitio
- generar mapa de páginas
- proponer sistema visual
- crear estructura base
- generar módulos funcionales
- dejar backlog y validación

## Módulos

### 1. Intent Intake

- idea
- audiencia
- objetivo
- tono
- industria

### 2. Website Planner

- sitemap
- core pages
- conversion pages
- support pages
- blog/resources si aplica

### 3. Content Engine

- headlines
- subheads
- sections
- CTA
- FAQs
- SEO blocks

### 4. Structure Engine

- layouts
- components
- forms
- cards
- grids
- dashboards si aplica

### 5. Delivery Engine

- responsive
- analytics
- SEO
- performance
- QA

## Modos

- landing builder
- SaaS site builder
- marketplace site builder
- community site builder
- content/media site builder
- corporate site builder

## Regla

La app debe producir sitios utilizables y escalables, no maquetas vacías.
