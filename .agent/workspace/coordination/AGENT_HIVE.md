# 🐝 AGENT HIVE — Espacio de Coordinación

> **Propósito:** Los agentes comparten información clave, learnings, soluciones y dudas para que todo el equipo aprenda y avance más rápido.

---

## 📢 Daily Updates

Cada agente publica aquí su avance diario al terminar su sesión.

### Formato de Update
```
## [YYYY-MM-DD] — [NombreAgente]
- **Tareas completadas:** TSK-001, TSK-002
- **Archivos tocados:** src/..., convex/...
- **Soluciones clave:** [descripción breve]
- **Bloqueos:** [si hay]
- **Próximo:** [qué sigue]
```

---

## ❓ Q&A — Consultas y Dudas

Espacio para que los agentes dejen preguntas que otros puedan responder.

### Formato de Pregunta

## Pregunta — [NombreAgente] — [YYYY-MM-DD]
**Contexto:** [qué estás haciendo]
**Pregunta:** [qué necesitas saber]
**Archivos:** [qué archivos están involucrados]

## 🧠 Knowledge Base — Learnings Compartidos

Soluciones, patrones y aprendizajes que cualquier agente puede reutilizar.

### Categorías
- **Auth:** Patrones de autenticación, JWT, sesiones
- **Payments:** MercadoPago, webhooks, suscripciones
- **Communities:** Creación, acceso, permisos
- **UI/UX:** Componentes reutilizables, estilos, animaciones
- **Convex:** Schema, mutations, queries, crons
- **Performance:** Optimizaciones, caching, lazy loading
- **Security:** Validaciones, sanitización, protección

### Formato de Entry
```
## [Categoría] — [Título] — [YYYY-MM-DD]
**Problema:** [qué pasaba]
**Solución:** [cómo se resolvió]
**Archivos:** [dónde está el código]
**Código clave:** [snippet si aplica]
**Notas:** [advertencias, edge cases]
```

## 📋 Reglas del Hive

1. **Escribir al terminar cada tarea** — No importa si es pequeña
2. **Ser conciso** — Bullet points, no essays
3. **Marcar consultas como resueltas** — Cuando se responda
4. **Categorizar knowledge** — Para que sea fácil encontrar
5. **No duplicar** — Buscar antes de preguntar
6. **Usar Aurora** — Consultar a Aurora antes de bloquearse

## 🚀 Delegación de Sub-Agentes

Cada agente puede delegar hasta **2 sub-agentes** por tarea compleja.

### Formato de Delegación
```
## Delegación — [AgentePrincipal] — [YYYY-MM-DD]
**Tarea padre:** TSK-XXX — [nombre]
**Sub-agente 1:** [rol] → [sub-tarea específica]
**Sub-agente 2:** [rol] → [sub-tarea específica]
**Estado:** ⏳ En curso | ✅ Completada
```
