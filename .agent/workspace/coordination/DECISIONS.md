# Decisions

## Regla

Toda decisión que cambie foco, marca, navegación o prioridades debe quedar registrada acá.

## Decisiones vigentes

### 2026-03-20

- Se invalida la documentación vieja de `.agent/skills` por duplicación y estados inconsistentes.
- El foco inmediato del producto es `Comunidad + Herramientas + Academia + Planes`.
- Instagram, ads avanzados y superficies satélite salen del centro narrativo del producto.
- El board único del proyecto pasa a ser `coordination/TASK_BOARD.md`.
- Se abre una línea P0 de infraestructura para cerrar runtime, config, datos, webhooks y auth server-side antes de seguir escalando integraciones.
- Convex se toma como backend objetivo y fuente de verdad para datos de negocio; `localStorage` queda relegado a cache, drafts y soporte local no crítico.
- El claim de tarea y la declaración de foco en `CURRENT_FOCUS.md` pasan a ser obligatorios antes de cualquier edición.
- `NON_NEGOTIABLES.md` queda como contrato de estabilidad del proyecto y no puede ser violado sin decisión explícita.
