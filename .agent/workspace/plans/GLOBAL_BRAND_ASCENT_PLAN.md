# Global Brand Ascent Plan

## Objetivo

Construir una marca mundial para la red social de inversionistas.

## Referencia de mercado

Según Kantar BrandZ 2025, las marcas más valiosas siguen siendo Apple, Google, Microsoft, Amazon y Nvidia, con Facebook e Instagram aún muy fuertes y TikTok/Instagram creciendo con velocidad.

Patrones comunes:
- categoría clara
- ecosistema
- distribución masiva
- hábito
- diferenciación
- consistencia
