# 📦 Skills Pendientes de Instalación

> Skills que Kimi recomendó pero requieren GitHub auth para instalar

## ⚠️ Requiere GitHub Auth

```bash
# Login primero
gh auth login

# Luego instalar
npx skills add vercel/ai-sdk --yes
npx skills add supabase/supabase-postgres-best-practices --yes
npx skills add better-auth/better-auth-best-practices --yes
npx skills add upstash/upstash-redis --yes
npx skills add upstash/upstash-workflow --yes
npx skills add authjs/authjs --yes
npx skills add resend/resend --yes
npx skills add getsentry/sentry --yes
```
