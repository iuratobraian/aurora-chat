# Aurora MCP Research Sources

## Fuentes Favoritas para Investigación de MCPs

### marketplaces
- **MCPMarket**: https://mcpmarket.com/es
  - Directorio: 27,518+ servidores MCP
  - Skills: 69,835+ habilidades de agente
  - Actualización: cada 2 horas
  
### blogs de referencia
- **Bright Data Blog**: https://brightdata.es/blog/ai
  - Top 10 MCP servers comparison
  - Reviews detallados de herramientas
  
### repositorios destacados
- **Awesome MCP Servers**: https://github.com/punkpeye/awesome-mcp-servers
  - Colección curada de cientos de MCPs
  
### news y updates
- **MCP Market News**: https://mcpmarket.com/es/news
  - Últimas novedades en MCP ecosystem

## Protocolo de Research

### Al Iniciar (PROTOCOLO_AMM)
1. Verificar mcpmarket.com/es/daily para MCPs trending
2. Revisar github.com/punkpeye/awesome-mcp-servers para nuevos MCPs
3. Verificar brightdata.es/blog/ai para reviews actualizada

### Integración de Nuevos MCPs
1. Research: Stars, GitHub activity,最后一次更新
2. Evaluación: Tools útiles, caso de uso, dependencies
3. Priorización: Crítica > Alta > Media
4. Agregar a connectors.json con readiness status
5. Log en AGENT_LOG.md

## Métricas de Calidad
- Stars mínimos para considerar: 500+
- Actualización última: < 6 meses
- Maintained: issues/PRs respondidos
