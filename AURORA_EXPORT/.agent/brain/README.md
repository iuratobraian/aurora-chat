# Project Brain

## Objetivo

Esta carpeta contiene la memoria local y la inteligencia acumulada del proyecto.

## Capas

- `datasets/`
  - datos destilados del trabajo del equipo
- `prompts/`
  - prompts de alto valor para la IA local
- `generated/`
  - artefactos regenerables
- `specs/`
  - definición del agente local y su operación
- `knowledge/`
  - criterio nativo de desarrollo, review, debugging y producción

## Regla

Todo agente debe tratar esta carpeta como memoria del proyecto, no como un cajón de archivos sueltos.
