# Delegation Playbook

## Objetivo

Delegar mejor que la media.

## Qué debe tener una buena delegación

- objetivo exacto
- contexto mínimo suficiente
- archivos a tocar
- archivos prohibidos
- criterio de salida
- riesgo principal

## Qué no se delega a modelos pequeños o baratos

- ambigüedad estratégica
- arquitectura crítica
- integración multiarchivo sin guía
- decisiones de seguridad o producción

## Regla

Si el agente podría interpretar la tarea de dos maneras muy distintas, la delegación está mal escrita.
