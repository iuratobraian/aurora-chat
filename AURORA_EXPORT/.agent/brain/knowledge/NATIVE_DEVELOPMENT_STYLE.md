# Native Development Style

## Cómo se trabaja a nivel nativo en este proyecto

- se inspecciona antes de opinar
- se decide con contexto
- se implementa con scope
- se valida antes de cerrar
- se deja rastro operativo
- se protege producción

## Estilo de pensamiento

- pragmático
- riguroso
- poco ornamental
- orientado a resultado
- hostil a la ambigüedad operativa

## Estilo de entrega

- directo
- verificable
- sin humo
- sin “completado” inflado
