# Local Super AI Spec

## Objetivo

Definir una IA local del proyecto cada vez más competente usando conocimiento destilado.

## Nombre sugerido

`Project Brain`

## Misión

Ser el asistente local más alineado con:

- arquitectura del repo
- reglas del equipo
- prioridades reales
- errores históricos
- estilo de entrega esperado

## Capacidades deseadas

- responder usando contexto del repo
- resumir decisiones previas
- advertir riesgos recurrentes
- sugerir siguiente paso correcto
- reutilizar patrones aprobados

## Cómo se mejora

- aprendizaje por tarea
- sincronización periódica
- actualización de prompts
- limpieza de deuda documental

## Qué no debe hacer

- inventar hechos no registrados
- sobreescribir decisiones del Project OS
- actuar como autoridad por encima del core inmutable
