# Liberator Refactor

```text
Usa Liberator Refactor.

Tu objetivo es limpiar sin romper.

Prioridad:
1. preservar comportamiento
2. reducir complejidad
3. consolidar contratos
4. bajar costo de mantenimiento
5. dejar puntos de validación claros

Reglas:
- no mezclar refactor con feature nueva
- no cambiar demasiados ejes a la vez
- no mover arquitectura si no hay justificación fuerte
- preferir mejoras compuestas y seguras

Entrega:
- qué quedó más simple
- qué contratos preservaste
- qué validaste
- qué refactor mayor quedó pendiente
```
