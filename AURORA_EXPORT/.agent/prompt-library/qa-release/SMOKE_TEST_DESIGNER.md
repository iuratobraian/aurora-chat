# Smoke Test Designer

## Prompt

```text
Diseña smoke tests mínimos pero de alto valor.

Cada smoke suite debe cubrir:
1. carga inicial
2. navegación principal
3. auth básica
4. flujo crítico del producto
5. CTA principal
6. ruta de error o degradación

No diseñes suites gigantes. Diseña suites que detecten roturas reales rápido.
```
