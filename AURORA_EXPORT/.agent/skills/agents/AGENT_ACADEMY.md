# Agent Academy

## Objetivo

Convertir agentes heterogéneos en una fuerza de trabajo profesional usando entrenamiento, contratos y funciones explícitas.

## Base principal

Todo agente debe comenzar por:

- `AGENT_CORE_INTELLIGENCE_BASE.md`
- `MODEL_INTELLIGENCE_ALIGNMENT_BASE.md`
- `HIGH_SIGNAL_FOUNDATION_STACK.md`
- `NOISE_REJECTION_CANON.md`
- `FRONTIER_CAPABILITY_STACK.md`
- `AURORA_CORE_MULTI_AGENT_ORCHESTRATION.md`

## Principio

No se intenta que todos los modelos hagan todo igual.

Se busca:

- elevar criterio operativo
- reducir errores previsibles
- asignarles el trabajo correcto
- compensar límites del modelo con estructura

## Niveles de entrenamiento

### Nivel 1: Operación básica

- leer Project OS
- reclamar task
- declarar focus
- respetar non-negotiables

### Nivel 2: Ejecución profesional

- usar specs y blueprints
- mantener source of truth
- evitar código zombie
- cerrar trabajo formalmente

### Nivel 3: Supervisión

- planear
- repartir
- detectar bloqueos
- revisar outputs de otros agentes
- decidir cuándo cortar o escalar

### Nivel 4: Maestria transversal

- entrar a verticales nuevas sin improvisar
- usar blueprints de web, mobile, juegos, IA y media
- producir specs, contratos y planes operativos
- adaptar el sistema a proyectos de magnitud

## Tracks de especializacion

- web y producto full stack
- mobile y super apps
- juegos mobile
- IA aplicada y automatizaciones
- video y media por prompt
- growth, operaciones y escalado
- inteligencia web, teardown y benchmark competitivo
