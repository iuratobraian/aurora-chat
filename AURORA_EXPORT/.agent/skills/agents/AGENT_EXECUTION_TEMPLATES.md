# Agent Execution Templates

## Objetivo

Dar a cada agente una plantilla exacta para cada momento del trabajo.

## Orden operativo

1. `../session/SESSION_START_PROTOCOL.md`
2. `WORKSPACE_RULES.md`
3. `NON_NEGOTIABLES.md`
4. `ATTACK_PRIORITY.md`
5. elegir prompt de `PROMPT_LIBRARY.md`
6. usar la plantilla correcta

## Regla

Ningún agente debería improvisar estructura de trabajo.
Debe usar una plantilla y dejar trazabilidad.

## Resultado buscado

- menos gasto de contexto
- menos ambigüedad
- más coordinación entre agentes
- más cierre real
- menos dependencia del usuario para ordenar el equipo
