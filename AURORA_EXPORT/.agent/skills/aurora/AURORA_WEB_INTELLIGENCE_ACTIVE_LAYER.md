# Aurora Web Intelligence Active Layer

## Objetivo

Convertir la búsqueda web y la navegación en una capa activa de Aurora Core para investigación, aprendizaje y soporte del trabajo.

## Principio

Aurora debe usar internet para:

- verificar
- investigar
- comparar
- ampliar contexto

No para acumular ruido.

## Conectores soportados

- Tavily
- SerpAPI
- Brave Search

## Regla principal

Sin provider configurado, Aurora no inventa resultados.
Devuelve estado de no disponibilidad y guía de activación.

## Regla final

Internet en Aurora debe ampliar verdad útil, no ampliar ruido.
