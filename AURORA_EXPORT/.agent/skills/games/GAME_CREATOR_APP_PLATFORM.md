# Game Creator App Platform

## Objetivo

Definir una futura app capaz de ayudar a crear juegos mobile de forma simple y potente.

## Lo que debe hacer

- convertir una idea corta en spec de juego
- generar blueprint
- elegir género base
- armar loop
- generar economía inicial
- crear backlog
- producir prompts y scaffolds

## Módulos

### 1. Idea Intake

- fantasy
- género
- target player
- plataforma
- tiempo de sesión

### 2. Game Design Engine

- core loop
- meta loop
- progression
- economy
- monetization

### 3. Build Engine

- template selector
- scene/module generator
- HUD generator
- analytics pack
- ad/IAP pack

### 4. Content Engine

- levels
- missions
- enemies
- rewards
- events

### 5. QA Engine

- checklist
- performance targets
- device tiers
- crash cases

## Modos

- hypercasual creator
- idle/tycoon creator
- puzzle creator
- runner creator
- strategy creator
- social sim creator

## Regla

La app debe producir juegos construibles, no ideas infladas.
