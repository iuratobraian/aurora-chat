# Game Dev Foundations

## Objetivo

Dar a los agentes base profesional para crear juegos mobile sin improvisar.

## Pilares

- core gameplay loop
- control feel
- progresion
- economia
- retencion
- live operations
- rendimiento

## Preguntas obligatorias

1. cual es el loop de 10 segundos
2. cual es la meta de 5 minutos
3. cual es la razon de volver mañana
4. como escala la dificultad
5. como monetiza sin romper la experiencia

## Sistemas basicos

- movimiento y input
- combate o accion principal
- UI/HUD
- inventario o progreso
- niveles o sesiones
- guardado
- analytics
- tienda o monetizacion si aplica

## Guardrails

- priorizar jugabilidad sobre cantidad de sistemas
- no inflar economia sin modelarla
- performance estable en dispositivos medios
- estados de error invisibles al usuario final

## Delivery

- primero vertical slice
- luego meta systems
- luego live ops
