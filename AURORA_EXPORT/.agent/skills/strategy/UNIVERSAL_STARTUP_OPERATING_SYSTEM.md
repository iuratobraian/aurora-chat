# Universal Startup Operating System

## Objetivo

Expandir este repo para que el equipo pueda ejecutar con nivel profesional proyectos de:

- apps web completas
- apps moviles multifuncionales
- juegos mobile
- backends y APIs
- bases de datos y datos operativos
- IA aplicada al producto
- automatizaciones
- video generado por prompt
- media, growth y operaciones

## Principio

No se busca saber un poco de todo.
Se busca tener un sistema para entrar a cualquier vertical con:

1. discovery claro
2. arquitectura suficiente
3. templates correctos
4. validacion
5. entrega profesional

## Regla transversal

Todo proyecto nuevo debe nacer con:

- spec del producto
