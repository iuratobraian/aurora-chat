# Prompt Library

## Objetivo

Usar prompts consistentes para elevar el entendimiento del usuario, la calidad de ejecución y la preparación para producción.

## Reglas

1. Elegir siempre un prompt base de la librería antes de arrancar.
2. Combinarlo con el Project OS, no usarlo aislado.
3. Usar prompts de perfil para agentes free.
4. Usar `ABSOLUTE_MODE.md` en trabajo sensible, blockers, infraestructura y producción.
5. Usar el perfil de supervisor para `gpt-5 nano` cuando opere como coordinador.
6. Usar `../session/SESSION_START_PROTOCOL.md` al inicio de cada sesión para entrar en modo operativo máximo del repo.
7. Usar `../prompt-library/production-mode/LIBERATOR_MODE.md` cuando se quiera máxima intensidad de ejecución sin romper límites ni resguardos.
8. Para trabajo específico usar:
   - `LIBERATOR_DEBUGGING.md`
   - `LIBERATOR_REFACTOR.md`
   - `LIBERATOR_INTEGRATION.md`
   - `LIBERATOR_PRODUCTION.md`
   - `LIBERATOR_COMMUNITY_SUPPORT.md`
   - `LIBERATOR_GAME_CREATION.md`
   - `LIBERATOR_WEB_CREATION.md`
   - `AURORA_CONVERSATION_MODE.md`
