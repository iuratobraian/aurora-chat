# Workflow: /apps

Este comando activa la especialización del agente para trabajar en la creación de nuevas apps.

## Pasos de Sincronización
1.  **Entrar al Directorio**: Navegar a `.agent/skills/creador_de_apps/`.
2.  **Leer SKILL**: Consumir `.agent/skills/creador_de_apps/SKILL.md`.
3.  **Auditoría de Mesa de Trabajo**:
    - Leer `workspace/TASK_BOARD.md`
    - Leer `workspace/CURRENT_FOCUS.md`
4.  **Selección de Tarea**: Elegir una tarea `pending` y marcarla como `claimed`.
5.  **Ejecución**: Aplicar el conocimiento de `infrastructure/` y `experience/` para avanzar en el desarrollo.

## Reglas
- Mantener la misma estética "Obsidian Ether" que TradePortal.
- Usar exclusivamente el workspace interno de `/creador_de_apps`.
- Documentar cada avance en `workspace/AGENT_LOG.md`.
