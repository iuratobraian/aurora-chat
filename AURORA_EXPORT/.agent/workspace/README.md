# Workspace

Este workspace contiene el estado operativo del proyecto.

## Carpetas

- `backlog/`
  - prioridades por horizonte
- `coordination/`
  - tablero, decisiones, handoffs y log de agentes
- `audits/`
  - diagnóstico de producto y técnico
- `plans/`
  - planes ejecutivos o entregables temporales

## Regla

El estado vivo del proyecto debe quedar aquí, no en archivos dispersos.
