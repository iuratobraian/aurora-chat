# Team Scorecard

## Propósito

Medir disciplina operativa y calidad de ejecución del equipo de agentes.

## Métricas

### Semanales

| Métrica | Objetivo |
|---|---|
| Tasks iniciadas con `CURRENT_FOCUS.md` completo | 100% |
| Tasks cerradas con `AGENT_LOG.md` actualizado | 100% |
| Cambios a archivos críticos con decisión registrada | 100% |
| Handoffs documentados sobre transferencias reales | 100% |
| Fallos de CI por disciplina operativa | 0 |
| Hardcodes de entorno detectados | 0 |
| Regresiones de estabilidad reportadas | 0 |

## Regla

Si un agente acumula violaciones de proceso, no toma archivos críticos hasta corregir disciplina operativa.
