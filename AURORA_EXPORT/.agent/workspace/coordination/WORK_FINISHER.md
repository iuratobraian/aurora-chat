# Work Finisher

## Propósito

Registrar trabajo que necesita cierre formal para no quedar huérfano.

## Uso

Si una tarea no puede pasar hoy a `done`, debe pasar por acá con salida clara.

## Plantilla

```md
## TASK-ID
- Estado final de sesión: blocked / cut / handoff
- Qué sí quedó hecho:
- Qué no quedó hecho:
- Qué falta para cerrar:
- Siguiente owner:
- Riesgo si se abandona:
```

## Entradas activas

Ninguna.
