# MCP and Automation Roadmap

## Fase 1

- GitHub MCP
- Browser / Playwright MCP
- Vercel MCP
- Sentry MCP
- acceso operativo a Convex

## Fase 2

- ticketing MCP: Linear o Jira
- diseño MCP: Figma
- documentación técnica adicional por proveedor

## Fase 3

- automatización de smoke tests
- validación visual
- release notes automáticas
- scorecards automáticos del equipo

## Regla

No instalar una herramienta si no se sabe:

- qué problema resuelve
- quién la usa
- qué riesgo agrega
- cómo se documenta su uso
