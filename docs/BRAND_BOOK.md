# TradePortal Brand Book

## 1. Brand Identity

### 1.1 Brand Name

- **Full name**: TradePortal
- **Short name**: TradePortal
- **Former names**: TradePortal, TradeShare
- **Logo usage**: Always "TradePortal" — no abbreviations

### 1.2 Tagline

> *"La plataforma donde los traders construyen credibilidad y comparten análisis profesional."*

Variants:
- "Tu comunidad de trading social en Latam."
- "Análisis de trading profesional, compartido."
- "Donde los mejores traders comparten sus estrategias."

### 1.3 Value Proposition

TradePortal es la plataforma de trading social para traders en Latam que quieren aprender de los mejores, compartir sus propias estrategias y construir historial verificado.

**Core promise**: Credibilidad a través de historial comprobable + acceso a análisis profesional + comunidad activa.

**Why TradePortal**:
- Construye tu historial: Cada operación compartida suma a tu reputación
- Aprende de los verificados: Solo traders con track record comprobable
- Crece con la comunidad: Feed rankeado por calidad, no solo por tiempo

**Key differentiators**:
1. Comunidad de trading con analysts verificados
2. Feed ranking por calidad y confianza
3. Señales de trading en tiempo real
4. Academia integrada con ruta de aprendizaje personalizada

## 2. Visual Identity

### 2.1 Logo

- Located at: `public/logo.svg`
- Primary: Logotipo TradePortal con rayo
- Colors: Primary blue (#3b82f6) on dark backgrounds
- Usage: Navbar, footer, PWA, SEO metadata

### 2.2 Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| Primary | `#3b82f6` | CTAs, links, highlights |
| Primary Dark | `#1d4ed8` | Hover states |
| Signal Green | `#22c55e` | Success, growth, buy signals |
| Alert Red | `#ef4444` | Errors, sell signals, warnings |
| Background Dark | `#0a0d12` | App background |
| Background Card | `#15191f` | Card surfaces |
| Text Primary | `#ffffff` | Headings |
| Text Secondary | `#9ca3af` | Body text |
| Text Muted | `#6b7280` | Captions, labels |

### 2.3 Typography

- **Headings**: System font stack with `font-black` (900 weight)
- **Body**: System font with `font-medium` (500 weight)
- **Monospace**: For numbers and code — use `font-mono`
- **Scale**: 8px base unit, modular scale for type hierarchy

### 2.4 Iconography

- **Library**: Material Symbols (Google Fonts)
- **Style**: Outlined, variable weight
- **Sizes**: 10px (micro), 14px (small), 18px (medium), 24px (large)

## 3. Tone of Voice

### 3.1 Core Attributes

| Attribute | Description |
|-----------|-------------|
| **Profesional** | Hablamos como traders experimentados — precisos, sin jerga innecesaria |
| **Directo** | Mensajes claros y accionables — sin rodeos |
| **Confiable** | Información verificada — no promovemos operacion sin fundamentos |
| **Comunitario** | Somos una comunidad — celebramos logros y compartimos conocimiento |

### 3.2 Do's and Don'ts

**DO**:
- "Empieza a compartir tu análisis hoy"
- "Señales verificadas con historial comprobable"
- "Tu próximo paso: [acción específica]"

**DON'T**:
- "¡GANA DINERO FÁCIL!" (tone: aggressive, misleading)
- "Opere ahora" without context
- "100% de accuracy" claims

### 3.3 Content Examples

**Onboarding**:
- "Bienvenido a TradePortal. La comunidad donde los traders profesionalizan sus operaciones."
- "Ve las mejores ideas de trading publicadas por nuestra comunidad."

**Feed post**:
- "[Nombre] comparte su análisis de [par] con [accuracy]% de precisión."
- "Discusión activa: [count] comentarios sobre esta estrategia."

**Dashboard**:
- "Tu próximo paso: [acción]"
- "Nivel: [X] — Sigue aprendiendo para subir de nivel."

## 5. Brand Architecture

### 5.1 Product Surfaces

- **Comunidad**: Feed social de análisis y estrategias
- **Señales**: Señales de trading con seguimiento
- **Academia**: Cursos y estrategias con ruta personalizada
- **Dashboard**: Resumen personal del día + próximo paso
- **Comunidades**: Grupos temáticos (crypto, forex, opciones)
- **Sala IA**: Asistente de soporte y recomendaciones

### 5.2 Feature Names

- Trust Layer: Sistema de confianza por autor y contenido
- FeedRanker: Algoritmo de ranking del feed
- LearningPath: Ruta de aprendizaje personalizada
- Signal Tracker: Seguimiento de señales de trading
- XP System: Gamificación del progreso del usuario
