# Research Benchmarks - TradePortal

## Overview

Dossiers de apps, plataformas y sitios de referencia para trading communities, social finance y creator platforms. Base comparativa para producto, growth y arquitectura.

## Metodología

- Fuentes: websites oficiales, blog de ingeniería, press releases, análisis de terceros
- Métricas: estimaciones públicas, reportes de empresas, análisis de mercado
- Última actualización: 2026-03-21

## 1. Trading Communities & Social Trading

### TradingView
**Type:** Social trading platform + charting
**URL:** tradingview.com

| Métrica | Dato |
|---------|------|
| Usuarios mensuales | ~15M |
| MAU activos | ~4M |
| Crecimiento YoY | 25% |
| Ingreso estimado | $200M+/año |
| Modelo | freemium + subscription |

**Features Core:**
- Charts interactivos (Pine Script)
- Social trading (ideas, scripts)
- Screener de mercados
- Alertas y notifications
- Mobile apps (iOS, Android)

**Features Social:**
- Ideas de trading públicas
- Comments y likes
- Seguidores de traders
- Copy trading (beta)
- Comunidades por ticker

**Insights Técnicos:**
- Frontend: JavaScript (charting custom)
- Backend: Cloud-native
- Real-time: WebSocket para quotes
- APIs públicas para data

**Patrones Transferibles:**
- Gamification del profile (badges, XP)
- Content rating (upvotes)
- Leaderboards semanales
- Trading challenges

**URL:** https://tradingview.com

## 6. Insights Transferibles

### Para TradePortal

1. **Gamification Universal**
   - Badges y XP
   - Leaderboards semanales
   - Streaks de engagement
   - Progress bars

2. **Social Proof**
   - Trust scores públicos
   - Performance tracks
   - Follower counts
   - Community size

3. **Revenue Model**
   - Freemium con tiers claros
   - Transaction fees (20% platform)
   - Premium features
   - Referrals y affiliates

4. **Content Formatting**
   - Templates consistentes
   - Rich media (images, video)
   - Threads para largo
   - Short posts para engagement

5. **Retention Loops**
   - Daily streaks
   - Weekly challenges
   - Monthly leaderboards
   - Push notifications
