## Top 5 — Specs Ejecutables

### TOP 5-A: MorningBriefing AI

**#42 — App de briefings diarios con IA**

### Concepto

"Tu briefing diario de mercados en 2 minutos. Análisis, alertas y oportunidades, generado por IA cada mañana."

### Por qué top 5

- **Score alto**: 4.3/5
- **Tiempo bajo**: 2-3 semanas
- **Synergy máxima**: alimentaría Dashboard existente
- **Revenue rápido**: subscription o freemium
- **Complemento directo**: ya existe `MorningBriefingCard.tsx` en codebase

### Spec Rápida

```typescript
interface Briefing {
  date: string;
  marketSummary: string;      // Overview del día
  topOpportunities: Opportunity[];
  alerts: Alert[];
  news: NewsItem[];
  watchlist: string[];       // tickers a seguir
}

interface Opportunity {
  ticker: string;
  direction: 'long' | 'short';
  reason: string;
  confidence: number;         // 0-100
  timeframe: string;
}
```

**Pantallas:**
- Feed de briefing (scroll)
- Detalle de oportunidad
- Config de alertas
- Watchlist

**Modelo:** Free (3 briefings/sem) + Pro ($5/mes, daily + alerts)

    nichePercentile: number;

  };

**Pantallas:**

### TOP 5-D: TradeJournal Pro

**#27 — Journal profesional de trading**

### Concepto

"Tu journal de trading inteligente. Registra operaciones, analiza patrones y mejora tu edge."

### Por qué top 5

- **Score**: 3.6/5
- **Synergy extrema**: mercado trading + retention alto
- **Diferenciador**: journal + IA + comunidad
- **Complemento**: Signals, Dashboard, TradeSim
- **Revenue**: subscription + data products

### Spec Rápida

```typescript
interface TradeEntry {
  id: string;
  date: number;
  
  setup: {
    ticker: string;
    direction: 'long' | 'short';
    entryPrice: number;
    exitPrice: number;
    size: number;
    stopLoss: number;
    takeProfit: number;
  };
  
  tags: string[];           // 'breakout', 'mean_reversion', etc.
  emotion: 'confident' | 'neutral' | 'fearful' | 'greedy';
  confidence: 1 | 2 | 3 | 4 | 5;
  
  result: {
    pnl: number;
    pnlPercent: number;
    duration: number;
    rrr: number;            // Risk:Reward ratio achieved
  };
  
  notes: string;
  screenshot?: string;
}
```

**Pantallas:**
- New entry (form rápido)
- Trade log (tabla filtrable)
- Analytics (P&L, win rate, RRR, emotion correlation)
- Pattern detection (AI)
- Journal feed (comunidad)

**Modelo:** Free (20 trades/mes) + Pro ($7/mes, unlimited) + Elite ($15/mes, AI analysis)
